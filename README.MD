用于进行同样结构JSON之间的互相转换。
同样结构指的是“结构相似”，比如JSON数据只包括2个维度的数组、JSON数据无数组等。

例子1：
```
{
    "a" :{
        "b": {
            "c":1
        }
    }
}
和下面的JSON属于同样结构
{
    "x1":1
}
```

例子2：
```
{
     "a": {
         "b": [
             {
                 "c": {
                     "d": {
                         "e": [
                             null,
                             -1,
                             4
                         ]
                     },
                     "f": [
                         -112,
                         -44
                     ]
                 }
             }
         ]
     }
 }

和下面的JSON属于同样结构
{
  "__DST__": {
    "x1": {
      "x2": [
        {
          "x3": [
            null,
            -1,
            4
          ]
        }
      ]
    },
    "x4": {
      "x5": [
        {
          "x5": [
            -112,
            -44
          ]
        }
      ]
    }
  }
}
```
